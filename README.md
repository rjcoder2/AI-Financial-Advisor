# AI Financial Advisor 💰

A Python GUI application that provides financial advice based on income, expenses, and financial goals.

---

## Pre-requisites

Before running the project, make sure the following are installed:

### Software
- Python 3.8 or higher
- Code Editor (VS Code / PyCharm)

### Python Libraries
- tkinter
- streamlit
- google-generativeai

### Basic Knowledge
- Python programming
- Basic financial concepts
- API usage

## Project Flow

The AI Financial Advisor works in the following steps:

1. User enters financial information such as income, expenses, and savings.
2. The system processes the financial data.
3. Financial analysis is performed to calculate savings and spending patterns.
4. The AI model (Gemini API) generates personalized financial advice.
5. Results and financial insights are displayed to the user through the interface.
