# AI Financial Advisor 💰

A Python GUI application that provides financial advice based on income, expenses, and financial goals.

---

## Pre-requisites

Before running the project, make sure the following are installed:

### Software
- Python 3.8 or higher
- Code Editor (VS Code / PyCharm)

### Python Libraries
- tkinter
- streamlit
- google-generativeai

### Basic Knowledge
- Python programming
- Basic financial concepts
- API usage
