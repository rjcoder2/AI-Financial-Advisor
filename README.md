# AI Financial Advisor 💰

An interactive **Python-based financial advisor application** that analyzes a user's income, expenses, and savings to generate financial insights and recommendations.
The application provides financial analysis, AI-generated advice, and visualizations to help users make better financial decisions.

---

## Features

* Analyze income, expenses, and savings
* Calculate financial metrics such as balance and savings rate
* Generate AI-based financial advice
* Visualize expense distribution using charts
* Simple and user-friendly interface

## Project Flow

The **AI Financial Advisor** works in the following steps:

1. The user enters financial information such as:

   * Income
   * Savings
   * Monthly expenses

2. The application processes the financial data.

3. Financial analysis is performed to calculate:

   * Remaining balance
   * Savings rate
   * Expense ratio

4. The AI module generates personalized financial advice based on the analysis.

5. The results, insights, and expense visualization are displayed to the user through the interface.
